## 🔧 配置说明

### 前端API地址配置

前端默认连接：`http://localhost:3000/api`

如果需要修改，创建 `.env` 文件（项目根目录）：
```env
VITE_API_BASE_URL=http://localhost:3000/api
```

### 后端CORS配置

后端已配置允许来自 `http://localhost:5173` 的请求。

如果需要修改，编辑 `backend/.env`：
```env
CORS_ORIGIN=http://localhost:5173
```

### 示例：连接合同上传功能

修改 `src/components/ContractAnalysis.tsx`：

```typescript
import { uploadContract, analyzeContract } from '../utils/api';

// 在handleUpload函数中：
const handleUpload = async (file: File) => {
  setAnalyzing(true);
  try {
    // 上传文件
    const result = await uploadContract(file);
    const contractId = result.data.contract.id;
    
    // 开始分析
    await analyzeContract(contractId);
    
    // 轮询检查分析结果
    const checkAnalysis = setInterval(async () => {
      const contract = await getContract(contractId);
      if (contract.data.contract.status === 'completed') {
        clearInterval(checkAnalysis);
        setAnalyzing(false);
        setShowResult(true);
        // 显示分析结果
      }
    }, 2000);
  } catch (error) {
    console.error('上传失败:', error);
    setAnalyzing(false);
  }
};
```

## ⚠️ 常见问题

### 问题1：CORS错误

**错误信息：**
```
Access to fetch at 'http://localhost:3000/api/auth/login' from origin 'http://localhost:5173' has been blocked by CORS policy
```

**解决方法：**
1. 检查后端 `.env` 中的 `CORS_ORIGIN` 配置
2. 确保后端服务器正在运行
3. 重启后端服务器

### 问题2：404错误

**错误信息：**
```
Failed to fetch
```

**解决方法：**
1. 确认后端服务器运行在 `http://localhost:3000`
2. 检查前端 `.env` 中的 `VITE_API_BASE_URL`
3. 查看浏览器控制台的完整错误信息

### 问题3：401未授权错误

**错误信息：**
```
401 Unauthorized
```

**解决方法：**
1. 重新登录获取新token
2. 检查token是否正确保存到localStorage
3. 检查后端JWT_SECRET配置

## 🎓 理解前后端通信

### 请求流程

```
用户操作（点击按钮）
    ↓
前端组件调用API函数
    ↓
API函数发送HTTP请求到后端
    ↓
后端处理请求（查询数据库）
    ↓
后端返回JSON响应
    ↓
前端更新界面显示结果
```

### 用C语言理解

```c
// 前端 = 客户端程序
void handleLogin() {
    // 发送HTTP请求（类似C中的socket通信）
    Response response = sendRequest("POST", "/api/auth/login", data);
    
    // 处理响应
    if (response.success) {
        saveToken(response.token);  // 保存token（类似保存配置）
        showMainPage();  // 显示主界面
    }
}

// 后端 = 服务器程序
void handleLoginRequest(Request req) {
    // 验证用户（类似数据库查询）
    User* user = findUser(req.username);
    
    if (verifyPassword(user, req.password)) {
        // 生成token（类似生成会话ID）
        char* token = generateToken(user->id);
        
        // 返回响应（类似send()发送数据）
        sendResponse(200, {token: token});
    }
}
```
