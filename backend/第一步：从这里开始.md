# 🎯 第一步：从这里开始

## 快速开始（5分钟）

### 步骤1：检查环境 ✅

打开终端，运行：

```bash
# 检查Node.js
node --version  # 应该显示 v18.x.x 或更高

# 检查npm
npm --version   # 应该显示 9.x.x 或更高

# 检查MongoDB（如果已安装）
mongosh --version
```

**如果没有安装：**
- Node.js: https://nodejs.org/ （下载LTS版本）
- MongoDB: `brew install mongodb-community` (macOS)

---

### 步骤2：进入项目目录 📁

```bash
cd /Users/nieweifan/Downloads/TrafficControlSystem/backend
```

---

### 步骤3：安装依赖 📦

```bash
npm install
```

等待安装完成（1-3分钟）

---

### 步骤4：配置环境变量 ⚙️

```bash
# 复制配置文件
cp env.example .env

# 编辑.env文件（使用你喜欢的编辑器）
# macOS:
open -e .env
# 或
nano .env
# 或使用VS Code:
code .env
```

**必须修改的内容：**

1. **生成JWT密钥**（在Node.js中运行）：
```bash
node
```
然后在Node.js REPL中：
```javascript
const crypto = require('crypto');
console.log('JWT_SECRET:', crypto.randomBytes(32).toString('hex'));
console.log('ENCRYPTION_KEY:', crypto.randomBytes(32).toString('hex'));
// 复制输出的值，按Ctrl+C退出
```

2. **将生成的值填入.env文件：**
```env
JWT_SECRET=你生成的64位十六进制字符串
ENCRYPTION_KEY=你生成的64位十六进制字符串
```

---

### 步骤5：启动MongoDB 🗄️

```bash
# macOS (如果使用Homebrew)
brew services start mongodb-community

# 验证是否运行
brew services list
# 应该看到 mongodb-community 状态为 started
```

---

### 步骤6：创建上传目录 📂

```bash
mkdir -p uploads
```

---

### 步骤7：启动服务器 🚀

**方法1：使用快速启动脚本（推荐）**
```bash
./快速启动.sh dev
```

**方法2：手动启动**
```bash
npm run dev
```

**成功标志：**
```
✅ MongoDB连接成功: 127.0.0.1
🚀 服务器运行在 http://localhost:3000
📝 环境: development
```

## ❓ 遇到问题？

### 常见问题：

1. **MongoDB连接失败**
   ```bash
   brew services start mongodb-community
   ```

2. **端口被占用**
   - 修改 `.env` 中的 `PORT=3001`

3. **模块找不到**
   ```bash
   rm -rf node_modules
   npm install
   ```

4. **权限问题**
   ```bash
   chmod +x 快速启动.sh
   ```
