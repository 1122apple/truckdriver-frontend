
### 方法2：查看MongoDB数据库

```bash
# 连接MongoDB

db.users.find().pretty()
```

应该能看到刚注册的用户数据。

## ⚠️ 常见问题

### 问题1：注册失败 - 用户名已存在

**解决方法：**
- 使用不同的用户名
- 或删除数据库中的旧数据：
  ```bash
  mongosh traffic_control
  db.users.deleteMany({})
  ```

### 问题2：CORS错误

**解决方法：**
- 确保后端 `.env` 中 `CORS_ORIGIN=http://localhost:5173`
- 重启后端服务器

### 问题3：页面无法切换

**解决方法：**
- 检查浏览器控制台是否有错误
- 确保所有文件都已保存
- 重启前端开发服务器
